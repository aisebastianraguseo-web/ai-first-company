# Smart Bookmark Manager - Product Overview

## Elevator Pitch

Ein AI-powered Bookmark Manager der URLs automatisch kategorisiert, sodass Wissensarbeiter ihre gespeicherten Links wiederfinden ohne manuelles Tagging oder Chaos.

## Target Users

1. **Wissensarbeiter** - Researcher, Developer, Designer die täglich 10+ neue Links sammeln und frustriert sind über Browser-Bookmark Chaos
2. **Power-User** - Menschen mit 200+ Bookmarks die aktuell nichts wiederfinden und manuelle Kategorisierung zu aufwändig finden

## Core Value

**"Save it, find it - ohne zu denken"** - Als einziger Bookmark Manager mit automatischer AI-Kategorisierung eliminieren wir den Overhead von manueller Organisation.

## MVP Scope

- Painless Add (Manual URL Entry)
- AI Kategorisierung (Claude API)  
- Smart Search (Text-based)
- Simple Organization (Categories, Filters)
- User Authentication (Clerk.dev)

## Success Definition

**User Activation:** User added >10 Bookmarks innerhalb 7 Tage  
**AI Quality:** User akzeptiert AI-Kategorien in >70% der Fälle  
**Retention:** User kommt zurück (DAU/MAU >20%)
